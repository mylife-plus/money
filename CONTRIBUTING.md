### Push Commit Message
```
#<issue-number> <⏳h of total work> <summarize the change> 
< - ⏳h of work description of changes via bullet points> 
For example: 
#42 ⏳2h commit Title
- ⏳2h I fixed this and that "

```

### Branch Naming
```
issue-<issue-number> 
example: issue-42
```

### Pull Request Title
```
#<issue-number> <summarize the change> 
example: 
#43 fix login bug 
```

### Pull Request Description
```
< - ⏳h of work description of changes via bullet points> 
For example: 
- ⏳2h I fixed this and that
```
